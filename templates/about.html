Tentang TomatoDoc AI

Sistem deteksi penyakit daun tomat berbasis Machine Learning untuk membantu petani mengidentifikasi masalah kesehatan tanaman secara dini dan akurat.

Teknologi yang Digunakan
Machine Learning

Random Forest Classifier dengan ekstraksi fitur computer vision untuk klasifikasi penyakit daun tomat.

Computer Vision

Analisis fitur HSV, statistik warna, dan tekstur gradien untuk identifikasi pola penyakit.

Backend

Flask web framework dengan Python untuk API dan pemrosesan gambar.

Frontend

Bootstrap 5, HTML5, CSS3, dan JavaScript untuk antarmuka yang responsif dan modern.

Cara Kerja Sistem
1. Upload Gambar

Pengguna mengupload foto daun tomat melalui antarmuka web

2. Preprocessing

Gambar diproses dan fitur visual diekstraksi menggunakan OpenCV

3. Prediksi ML

Model Random Forest menganalisis fitur dan memprediksi penyakit

4. Hasil & Rekomendasi

Sistem menampilkan diagnosis dan rekomendasi penanganan
